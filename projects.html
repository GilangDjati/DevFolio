A selection of four key projects I developed and documented, each highlighting the challenges, solutions, and measurable outcomes.

Disaster Data Management Application

Challenge: Building a secure and real-time disaster data management system. Solution: Developed a web application using React.js, Redux, Express.js, and SQLite to handle user authentication, victim data management, and location-based real-time updates. Result: Successfully tested with full frontend–backend integration and deployed with video documentation on Vercel.

  • Stack: React.js, Redux, Express.js, SQLite, Tailwind CSS, SweetAlert2.

Real-Time Air Quality Monitoring System

Challenge: Providing real-time air quality data during forest fire incidents. Solution: Created a web app using React.js, Redux, Tailwind CSS, and the OpenWeatherMap API to monitor air pollution levels and deliver health recommendations. Result: Successfully deployed on Vercel with complete video documentation.

  • Stack: React.js, Redux, Tailwind CSS, OpenWeatherMap API.

Digital Literacy Awareness – PKM-PM

Challenge: Educating elementary school students on responsible gadget usage. Solution: Organized digital literacy sessions, interactive Q&A, coding games via Code.org, and end-to-end documentation. Result: Executed successfully with active student participation and strong teamwork across facilitators.

  • Stack: Public Speaking, Team Collaboration, Event Management, Documentation.

E-Commerce Mobile Application for SMEs

Challenge: Helping small businesses manage operations digitally. Solution: Built an Android application using Java and SQLite to manage products, transactions, and customers with role-based access.
